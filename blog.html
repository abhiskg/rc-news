01. What is the difference between var, let, and const?

var declarations are globally scoped or function/locally scoped. var variables can be re-declared and re-assign. We can declare var variables after defining them.

let is block scoped. Let can't be re-declared but can be re-assign.

const is block scoped. Can't be re-declared and re-assign. But can update properties of object and array.

While var and let can be declared without being initialized, const must be initialized during declaration.

02. What is the difference between regular function and arrow function?

Regular function can have duplicate named parameter but Arrow functions can never have duplicate named parameters.

Unlike regular functions, arrow functions do not have their own this. In arrow function this keyword is always bound to the closest non-arrow parent function.

Regular function has hoisting effect that means regular function can be called before declaring but in arrow function that is not possible.

03. Why should you use template string?

Template string lets us insert/interpolate variables and expressions into strings without needing to concatenate.

Another great feature of template strings is that we can have strings with multiple lines for better code readability. This cannot be done in an easy way with the old style of strings.

Template strings allow us to nest regular strings or template strings within template strings.
